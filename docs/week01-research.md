# Week 01 – Tìm hiểu Git và Godot cơ bản

## Nội dung tìm hiểu
- Git và GitHub dùng để quản lý mã nguồn
- Cách tạo repository và commit code
- Làm quen với Godot Engine
- Hiểu khái niệm Scene và Node

## Nguồn tham khảo
- https://roadmap.sh
- https://docs.godotengine.org
- Một số video hướng dẫn Godot trên YouTube

## Những gì đã hiểu

### Git / GitHub
- Git dùng để quản lý phiên bản code
- Commit giúp lưu lại các thay đổi
- Repository là nơi chứa project trên GitHub

### Godot Engine
- Scene giống như một màn hình hoặc một phần của game
- Node là thành phần cơ bản tạo nên scene
- Có thể ghép nhiều node để tạo nhân vật hoặc vật thể trong game

Ví dụ:
- Node2D dùng cho object 2D
- Sprite2D dùng để hiển thị hình ảnh
- CollisionShape2D dùng để tạo va chạm

## Ghi chú
Godot sử dụng cấu trúc node rất giống cây (tree structure).  
Điều này giúp tổ chức game object dễ dàng hơn.
